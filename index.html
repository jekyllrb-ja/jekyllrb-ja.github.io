プレーンテキストを静的な Web サイトやブログに変換します。

シンプル

データベースや、コメント管理や、インストールのためのやっかいなアップデートは不要です。 — あなたのコンテンツに集中できます。

基本的な使い方 →

静的

Markdown (または Textile)、 Liquid や HTML & CSS が使われます。 静的サイトの展開のための準備ができて出てきます。

Jekyll テンプレートガイド →

Blog のような

パーマリンク、カテゴリ、ページ、ポスト、そしてカスタムレイアウトはここの全てのファーストクラス市民です。

あなたのブログの移行 →

起動から実行まで数秒です。

Quick-start Instructions

~ $ gem install jekyll

~ $ jekyll new my-awesome-site

~ $ cd my-awesome-site

~/my-awesome-site $ jekyll serve

# => http://localhost:4000 を見てください
